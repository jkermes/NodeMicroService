# NodeMicroService

### Npm install :
	
- Into back directory :

	npm install
	
- Into front directory :

	npm install
	
- Into back/test directory :

	npm install

### To launch the server :

	node --use-strict back/server/index.js
	
### To launch client :

	php -S localhost:8080 -t front/

### To launch webpack:

	node_modules/.bin/webpack -w --config

### Folder structure :

	├ front (react/css)
	├ back (server)
